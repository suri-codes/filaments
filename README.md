# Filaments

My personal-knowledge-system, with deeply integrated task tracking and
long term goal planning capabilities.

This is a continuation of my previous project, [TARS](https://github.com/suri-codes/TARS).

## Name Origin

Derived from [galaxy filaments](https://en.wikipedia.org/wiki/Galaxy_filament),
with clusters being notes, and filaments being the links between them.


## AI Policy

All the code for this project is hand written, (I'm counting LSP
completions as hand-written).  I haven't and don't plan to use "Claude
Code" or adjacent tools for this project.

However, I have and will continue to use the chat interface for Claude
to ideate on architectural decisions, debugging obscure errors, etc.

If I do end up "vibe-coding" (extremely unlikely), I will update this section of the README
to reflect that, as I believe there is some worth in you knowing how
this code came to be.
